⚡ 一键切换

# 仅用本地 Ollama
set LLM_PROVIDER=ollama

# 仅用云端 SiliconFlow
set LLM_PROVIDER=siliconflow
set SILICONFLOW_KEY=你的密钥

# 自动模式（优先本地，失败回云端）
set LLM_PROVIDER=auto

快速开始

1

安装 Ollama 并拉取模型

从 ollama.ai 下载安装，然后拉取模型：

ollama pull qwen2.5:7b
ollama pull nomic-embed-text
2

安装 Obsidian 插件

将 kb-plugin/ 复制到你的 vault：

<Vault>/.obsidian/plugins/kb-plugin/

然后在 设置 → 第三方插件 中启用"知识库助手"

3

启动后端服务

cd kb-backend
pip install -r requirements.txt
python main.py
4

开始使用

点击侧边栏机器人图标 🤖，输入问题，AI 基于你的知识库回答！
