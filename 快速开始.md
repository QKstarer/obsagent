# 快速开始

## 3 步安装

### 第 1 步：安装前置软件

| 软件 | 下载地址 | 说明 |
|------|----------|------|
| Obsidian | https://obsidian.md | 笔记软件 |
| Python | https://www.python.org/downloads/ | 编程语言，安装时勾选"Add to PATH" |
| Ollama | https://ollama.com/download | 本地 AI 模型运行环境 |

### 第 2 歌：运行安装脚本

双击 `install.bat`，脚本会自动：
- 检查前置软件
- 安装 Python 依赖
- 下载 AI 模型（约 5GB，首次需要）

### 第 3 步：部署插件

将 `kb-plugin` 文件夹复制到：
```
<Vault路径>/.obsidian/plugins/kb-plugin/
```

在 Obsidian 中启用插件：设置 → 第三方插件 → 知识库助手

---

## 日常使用

### 启动

1. 双击 `start-backend.bat` 启动后端
2. 打开 Obsidian，点击左侧机器人图标 🤖

### 关闭

1. 关闭后端命令行窗口
2. 或按 `Ctrl+C` 停止服务

## 常见问题

**Q: Ollama 下载慢？**
A: 使用国内镜像或科学上网

**Q: Python 找不到？**
A: 安装 Python 时勾选 "Add to PATH"，或手动添加环境变量

**Q: 模型下载失败？**
A: 检查网络连接，或手动运行 `ollama pull nomic-embed-text`

**Q: 后端启动报错？**
A: 检查端口 8000 是否被占用，或修改配置中的端口号
